GAMEPLAY INSTRUCTIONS

CONTROLS/KEYBINDINGS:

R 						- reload
Hold shift 					- run
WASD 						- move controls
Right mouse 					- aim
Hold down right mouse while moving mouse 	- rotate player
Left mouse 					- shoot
Space bar 					- weapon selection HUD (work in progress)

THINGS TO CONSIDER WHEN TESTING:

1) Medkit air-dropped once player's health falls below 50%
2) Enemy has 80% chance of moving to a new position when being shot at (ie. EVADE state)
3) Blood oozes from fallen enemy when dead
4) Quit button in game-over screen only works when project built into .exe file
